# Use TypeScript for Backend Development

Status: Accepted
Date: 2023-05-15
Tags: backend, typescript, nodejs

## Context

We need to choose a programming language for our backend services that provides type safety and good developer experience.

## Decision

We will use TypeScript for all backend development.

## Consequences

- Improved type safety and developer experience
- Better IDE support and code completion
- Slight learning curve for developers not familiar with TypeScript
- Need for compilation step before running code
